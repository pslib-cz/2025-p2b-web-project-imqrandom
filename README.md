⚽ World Cup 2026 – Web Project
O projektu

Tento projekt je zaměřen na vytvoření moderní responzivní webové stránky na téma FIFA World Cup 2026.
Web bude sloužit jako informační a vizuálně atraktivní prezentace největší fotbalové události světa, která se uskuteční v roce 2026 v USA, Kanadě a Mexiku.

Cílem projektu je propojit moderní webdesign, responzivní layout a interaktivní prvky do přehledného a dynamického webu vytvořeného metodou mobile-first.

🎯 Cíl webu
představit turnaj World Cup 2026,
zobrazit informace o hostitelských zemích a stadionech,
nabídnout přehled týmů a skupin,
vytvořit galerii a vizuálně atraktivní obsah,
procvičit tvorbu moderního responzivního webu pomocí HTML, CSS a JavaScriptu.
👥 Cílová skupina
fanoušci fotbalu,
sportovní nadšenci,
mladší uživatelé sledující sportovní akce online,
návštěvníci hledající informace o mistrovství světa.
📱 Návrh webu

Web bude navržen ve Figmě s důrazem na:

moderní sportovní vzhled,
výraznou typografii,
práci s fotografiemi stadionů a hráčů,
responzivní design od mobilu po desktop,
přehlednou navigaci,
animace a interaktivní prvky.
🧩 Struktura webu
1. Home Page
úvodní hero sekce,
countdown do začátku turnaje,
základní informace o World Cup 2026,
CTA tlačítka.
2. Teams / Groups
přehled týmů,
skupiny turnaje,
statistiky a informace.
3. Stadiums / Host Cities
galerie stadionů,
informace o hostitelských městech,
interaktivní layout.
4. Gallery
fotogalerie vytvořená pomocí CSS Grid,
možnost otevření obrázků pomocí LightGallery nebo Swiperu.
5. Contact / About
kontaktní sekce,
informace o projektu.
